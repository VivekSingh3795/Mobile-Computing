# Simple Database App 📱

A lightweight Android application built with **Kotlin** and **Room Persistence Library** (SQLite) to demonstrate basic CRUD (Create, Read, Delete) operations.

## ✨ Features
* **Insert Data:** Add user names into a local SQLite database.
* **Persistent Storage:** Data stays saved even if the app is closed.
* **Display List:** View all stored entries with their unique IDs.
* **Clear Database:** Wipe all data with a single click.

## 🛠️ Tech Stack
* **Language:** [Kotlin](https://kotlinlang.org/)
* **Database:** [Room](https://developer.android.com/training/data-storage/room) (SQLite abstraction)
* **Concurrency:** [Kotlin Coroutines](https://kotlinlang.org/docs/coroutines-overview.html)
* **UI:** XML Layouts (View System)

## 🚀 How to Run
1. Clone this repository: `git clone https://github.com/YOUR_USERNAME/SimpleDatabaseApp.git`
2. Open the project in **Android Studio**.
3. Let Gradle sync and download dependencies.
4. Run the app on an Emulator or physical device (API 24+).
